# Shared content brand context

This directory is the project-scoped source of truth used by the imported content studio skills.

Run `$content-setup` to create or update:

- `BRAND.md` — positioning, audience, differentiators, product context
- `VOICE.md` — tone, vocabulary, rhythm, banned phrases, channel overrides
- `VISUAL.md` — colors, typography, logo usage, image direction

Keep real brand samples in `voice-samples/`. Do not store secrets here.
