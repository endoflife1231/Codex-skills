# Unlighthouse setup for Codex

The `$seo-unlighthouse` skill is already installed and requires no API key. It needs Node.js, `npx`, a compatible browser runtime, and permission to crawl the target.

Use the local preflight in [the central Codex integration guide](../../../CODEX_INTEGRATIONS.md). Start with a small route limit, keep output in a project artifact directory, and do not treat local lab measurements as production field data.
