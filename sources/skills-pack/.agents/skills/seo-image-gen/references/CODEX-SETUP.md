# Image-generation MCP setup for Codex

The `$seo-image-gen` skill, scripts, presets, and references are already installed. Configure `nanobanana-mcp` using [the central Codex integration guide](../../../CODEX_INTEGRATIONS.md).

Before generation, confirm the MCP tool is visible, the target model/package capability is current, and the user approves any paid use. The bundled direct-API scripts are an explicit fallback and require `GOOGLE_AI_API_KEY` in the process environment.
