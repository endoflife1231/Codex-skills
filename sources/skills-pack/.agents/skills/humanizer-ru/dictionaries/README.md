# Dictionaries

Расширенные справочники для скилла. SKILL.md ссылается на них при работе со специфическими паттернами или предметными областями.

## Файлы

| Файл | Зачем |
|---|---|
| [calques.md](calques.md) | Расширенный словарь калек с английского (4 класса, 50+ примеров). Используется правилами 2.8 и 7.* |
| [whitelist-it.md](whitelist-it.md) | ИТ-термины, которые НЕ заменять: оркестрация, эмбеддинги, идемпотентность, контейнеризация и др. |
| [whitelist-finance.md](whitelist-finance.md) | Финансовые и юридические термины, которые НЕ заменять: эмиссия, цессия, факторинг, секъюритизация и др. |
| [whitelist-medicine.md](whitelist-medicine.md) | Медицинские термины: анамнез, патогенез, иммуногистохимия. Плюс правило для юридически значимых дисклеймеров. |

## Когда какой загружать

Скилл при запуске определяет предметную область из:
- Явного указания пользователя («это про ИТ», «текст про финансы»)
- Пресета (`docs` чаще ИТ, иногда медицина)
- Содержимого текста (наличие профильных терминов)

Загружает соответствующий whitelist и применяет как `context_exceptions` для текущей сессии.

## Контрибуция

PR с расширением словарей приветствуются. Формат для калек: `Калька | Источник | Контекст | По-русски`. Формат для whitelist'ов: список терминов с краткими комментариями.

Не добавляйте бытовые слова — только специализированные термины с нормативным значением.
