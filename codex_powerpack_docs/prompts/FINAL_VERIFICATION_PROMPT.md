# Финальная верификация релиза

1. Распакуй архив в чистый временный каталог.
2. Установи Minimal.
3. Запусти verify.
4. Удали пакет.
5. Сравни проект с исходным состоянием.
6. Повтори для Standard.
7. Проверь режим без Graphify.
8. Проверь режим без Caveman.
9. Проверь dry-run.
10. Проверь повторную установку.
11. Проверь существующий AGENTS.md.
12. Проверь hashes.
13. Проверь отсутствие секретов.
14. Проверь состав архива.
15. Выдай PASS или FAIL.

Ошибки сначала зафиксируй в отчёте. Не исправляй молча.
