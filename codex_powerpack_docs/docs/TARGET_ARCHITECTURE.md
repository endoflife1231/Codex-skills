# Целевая архитектура

## Слои

1. Core policy — короткие постоянные правила.
2. Agents — специализированные роли.
3. Skills — загружаемые процессы.
4. Caveman — политика видимого вывода.
5. Graphify — анализ структуры проекта.
6. Installer — установка и сопровождение.
7. Profiles — состав компонентов.
8. Manifests — версии, источники и hashes.
9. Licenses — лицензии и notices.

## Поток сложной задачи

```text
Запрос
  ↓
Главный агент
  ↓
Explorer + Graphify
  ↓
Planner
  ↓
Implementer
  ↓
Tester + Reviewer + Security Reviewer
  ↓
Исправления
  ↓
Финальная проверка
  ↓
Caveman для пользовательского отчёта
```

## Источники истины

1. Исходный код.
2. Тесты и фактическое выполнение.
3. Конфигурация.
4. Документация проекта.
5. Graphify.
6. Предположения агента.
