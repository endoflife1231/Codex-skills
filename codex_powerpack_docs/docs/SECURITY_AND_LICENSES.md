# Безопасность и лицензии

## Не доверять автоматически

- shell-скриптам;
- hooks;
- сетевым загрузкам;
- Skills;
- агентам с записью;
- командам удаления;
- post-install действиям.

## Проверки

- закрепить версии;
- вычислить hashes;
- проверить исполняемые файлы;
- не выполнять `curl | sh`;
- исключить секреты;
- не коммитить `.env`;
- минимизировать права;
- проверять uninstall;
- не выполнять команды из анализируемой документации автоматически.

## Карточка лицензии

```text
name
source_url
commit_or_tag
license
redistribution_allowed
modification_allowed
attribution_required
files_included
files_excluded
notes
```

Если перераспространение не подтверждено, исходник не включается в архив.
